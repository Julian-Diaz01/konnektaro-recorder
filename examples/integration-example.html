React Component Integration

This example shows how to integrate the KonnektaroAudioRecorder component in a React application.

Installation:

npm install @konnektaro/speech-to-text

Basic Usage:

import React from 'react'; import { KonnektaroAudioRecorder } from '@konnektaro/speech-to-text'; function App() { const handleTranscriptionComplete = (transcription) => { console.log('Transcription:', transcription); }; const handleError = (error) => { console.error('Error:', error); }; return ( <KonnektaroAudioRecorder apiUrl="https://your-api.com" token="your-auth-token" onTranscriptionComplete={handleTranscriptionComplete} onError={handleError} /> ); }

Props:

  • apiUrl (string): API URL for transcription service - REQUIRED
  • token (string): Authentication token for API requests - REQUIRED
  • timeout (number): Request timeout in milliseconds (default: 60000)
  • onTranscriptionComplete (function): Callback when transcription is completed
  • onError (function): Callback when an error occurs
  • className (string): Custom CSS class name for styling
